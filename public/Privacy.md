# プライバシーポリシー

最終更新日: 2026-04-18

ARKHAM (Adventure Routing & Keeper Handling Assistant Manager)(以下「本ツール」といいます)は、利用者のプライバシーを尊重し、以下のとおり個人情報および利用情報の取り扱い方針を定めます。

## 1. 基本方針

- 本ツールはクライアントサイドで動作するブラウザアプリケーションです。
- 利用者がシナリオ編集・進行のために入力した情報(以下「利用者データ」といいます)は、原則として利用者のブラウザ内にのみ保存され、本ツールの提供者のサーバーに送信・保存されることはありません。

## 2. 本ツールがブラウザ内に保存する情報

本ツールは、利用体験の向上のため、ブラウザのローカルストレージに以下の情報を保存します。サーバーには送信されません。

- シナリオデータ(ノード、エッジ、変数、キャラクター、要素 等)
- 表示設定(テーマ、言語、エッジスタイル 等)
- キャンバスのビューポート(拡大率、表示位置)
- デバッグモードの有効/無効状態

保存された情報は、利用者がブラウザのキャッシュ・履歴を削除する、プライベートブラウジングを終了する等の操作により消去されます。重要なデータは、本ツールの「保存」機能から JSON ファイルとして書き出して保管してください。

## 3. 利用者データの送信

- 本ツールは、利用者データ(シナリオ本文、キャラクター名、変数値等)を提供者のサーバーや第三者に送信しません。
- 本ツールの「保存」機能は、利用者のローカル端末への JSON ファイルダウンロードであり、ネットワーク通信は発生しません。
- 本ツールの「読込」機能は、利用者がローカル端末から選択した JSON ファイルの読み取りであり、ネットワーク通信は発生しません。

## 4. 外部サービスの利用

本ツールは、以下の外部サービスを利用しています。これらのサービス利用時には、利用者のブラウザから各サービス事業者へ情報が送信されることがあります。

### 4.1 Google Fonts

- 本ツールは、タイトル表示用のフォント(Cinzel Decorative)を Google Fonts から読み込みます。
- フォントの読み込み時、IP アドレス・ユーザーエージェント・リファラ等が Google LLC に送信されます。
- 詳細は [Google プライバシーポリシー](https://policies.google.com/privacy) を参照してください。

### 4.2 Google Analytics

- 本ツールは、利用状況の分析のため Google Analytics (GA4) を使用しています(計測 ID: G-6CDFLQRTER)。
- Google Analytics は Cookie を利用し、利用者の IP アドレス、ブラウザ情報、訪問ページ、滞在時間等を収集します。
- 収集された情報は匿名で処理され、個人を特定する目的では使用されません。
- Google Analytics によるトラッキングを無効化したい場合は、[Google Analytics オプトアウト アドオン](https://tools.google.com/dlpage/gaoptout) をインストールしてください。
- 詳細は [Google Analytics の利用規約](https://marketingplatform.google.com/about/analytics/terms/jp/) および [Google プライバシーポリシー](https://policies.google.com/privacy) を参照してください。

## 5. Cookie について

- 本ツール自体は Cookie を独自に発行しませんが、前述の Google Analytics により第三者 Cookie が設定される場合があります。
- 利用者はブラウザ設定により、Cookie の受け入れを拒否したり、既存の Cookie を削除したりすることができます。
- Cookie を無効にした場合でも、本ツールの主要機能は利用できます。

## 6. アクセスログ

- 本ツール自体はアクセスログをサーバー側で取得しません。
- 本ツールをホスティングするサービス(GitHub Pages 等)のログについては、当該ホスティング事業者のプライバシーポリシーに従います。

## 7. 児童のプライバシー

- 本ツールは、13 歳未満の児童からの個人情報の収集を意図していません。
- 保護者の方で、お子様が本ツールを通じて情報を送信した可能性があると判断された場合は、後述のお問い合わせ先までご連絡ください。

## 8. セキュリティ

- 利用者データはブラウザのローカルストレージに保存されるため、同一ブラウザ・同一端末を利用する第三者から閲覧される可能性があります。共有端末でのご利用の際は、利用後に JSON 保存・ブラウザデータの消去等を行ってください。
- 提供者は、可能な限り安全な方法で本ツールを提供するよう努めますが、完全なセキュリティを保証するものではありません。

## 9. プライバシーポリシーの変更

- 法令の改正や本ツールの機能変更等に伴い、本ポリシーの内容を予告なく変更することがあります。
- 変更後のポリシーは、本ツール上に掲示された時点から効力を生じるものとします。

## 10. お問い合わせ

本ポリシーに関するお問い合わせは、下記のリポジトリの Issue からお願いします。

- GitHub: https://github.com/c-ardinal/ARKHAM

# Privacy Policy

Last updated: 2026-04-18

ARKHAM (Adventure Routing & Keeper Handling Assistant Manager) ("the Tool") respects the privacy of its users. This Policy sets forth how personal and usage information is handled.

## 1. Basic Policy

- The Tool is a client-side browser application.
- Information entered by the User for scenario editing and play ("User Data") is, in principle, stored only in the User's browser and is not transmitted to or stored on the Provider's servers.

## 2. Information Stored in the Browser

For an improved user experience, the Tool stores the following information in browser local storage. None of it is transmitted to any server.

- Scenario data (nodes, edges, variables, characters, elements, etc.)
- Display settings (theme, language, edge style, etc.)
- Canvas viewport (zoom level, position)
- Debug mode enabled/disabled state

The information stored above is erased when the User clears the browser cache or history, ends a private browsing session, or performs similar actions. We recommend exporting important data as JSON files using the Tool's "Save" feature.

## 3. Transmission of User Data

- The Tool does not transmit User Data (scenario content, character names, variable values, etc.) to the Provider's servers or to any third party.
- The "Save" feature downloads a JSON file to the User's local device; no network transmission occurs.
- The "Load" feature reads a JSON file selected by the User from their local device; no network transmission occurs.

## 4. Use of External Services

The Tool uses the following external services. When these services are used, information may be transmitted from the User's browser to the respective service provider.

### 4.1 Google Fonts

- The Tool loads a font for title display (Cinzel Decorative) from Google Fonts.
- When the font is loaded, the User's IP address, user agent, referrer, and similar information are transmitted to Google LLC.
- For details, please refer to the [Google Privacy Policy](https://policies.google.com/privacy).

### 4.2 Google Analytics

- The Tool uses Google Analytics (GA4) to analyze usage (Measurement ID: G-6CDFLQRTER).
- Google Analytics uses cookies and collects the User's IP address, browser information, pages visited, duration of stay, and similar data.
- The information collected is processed anonymously and is not used to identify individuals.
- If you wish to disable tracking by Google Analytics, please install the [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout).
- For details, please refer to the [Google Analytics Terms of Service](https://marketingplatform.google.com/about/analytics/terms/us/) and the [Google Privacy Policy](https://policies.google.com/privacy).

## 5. Cookies

- The Tool itself does not issue its own cookies; however, third-party cookies may be set by Google Analytics as described above.
- You may refuse cookies or delete existing cookies through your browser settings.
- The Tool's core functionality remains available even if cookies are disabled.

## 6. Access Logs

- The Tool itself does not collect access logs on a server.
- Logs on the hosting service where the Tool is deployed (such as GitHub Pages) are handled according to that hosting provider's privacy policy.

## 7. Children's Privacy

- The Tool is not intended to collect personal information from children under the age of 13.
- If a parent or guardian believes their child has transmitted information through the Tool, please contact us using the information below.

## 8. Security

- Because User Data is stored in browser local storage, it may be visible to third parties who use the same browser on the same device. When using a shared device, please save your data as JSON and clear browser data after use.
- The Provider endeavors to provide the Tool by secure means but does not guarantee complete security.

## 9. Changes to This Privacy Policy

- This Policy may be changed without prior notice due to legal amendments or feature changes of the Tool.
- The amended Policy shall take effect upon posting within the Tool.

## 10. Contact

For inquiries regarding this Policy, please use the Issues on the repository below.

- GitHub: https://github.com/c-ardinal/ARKHAM
